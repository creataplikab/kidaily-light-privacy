Politique de Confidentialité - Kidaily Light

Dernière mise à jour : 28 septembre 2025
Version : 1.0.5

1. Introduction

Kidaily Light est une application mobile dédiée à l'évaluation basique du développement de l'enfant. Cette politique de confidentialité explique comment nous collectons, utilisons et protégeons les informations personnelles des utilisateurs de notre application.

Important : Notre application est conçue pour les enfants de 0 à 36 mois et leurs parents/tuteurs. Nous nous engageons à protéger la vie privée des enfants conformément aux réglementations en vigueur, notamment le RGPD et la COPPA.

2. Informations que nous collectons

2.1 Informations sur l'enfant

  • Date de naissance : Pour déterminer l'âge et adapter les questions d'évaluation
  • Sexe : Pour personnaliser les recommandations (optionnel)
  • Réponses aux questions : Pour calculer les scores d'évaluation
  • Résultats d'évaluation : Scores par domaine de développement

2.2 Informations techniques

  • Préférences de langue : Français, Anglais, Allemand, Italien
  • Données d'utilisation : Temps passé dans l'application, sections visitées
  • Informations sur l'appareil : Type d'appareil, version du système d'exploitation

2.3 Données publicitaires (AdMob)

  • Identifiants publicitaires : Pour afficher des publicités personnalisées
  • Données de performance : Pour optimiser l'affichage des publicités

3. Comment nous utilisons vos informations

3.1 Objectifs principaux

  • Évaluation du développement : Analyser les réponses pour fournir des scores et recommandations
  • Personnalisation : Adapter les questions selon l'âge de l'enfant
  • Amélioration de l'application : Analyser l'utilisation pour optimiser l'expérience utilisateur
  • Support multilingue : Afficher l'interface dans la langue choisie

3.2 Monétisation

  • Publicités : Affichage de publicités via Google AdMob
  • Analytics : Mesure de l'engagement utilisateur

4. Partage d'informations

Nous ne vendons jamais vos données personnelles.

4.1 Partenaires de confiance

  • Google AdMob : Pour l'affichage de publicités (conforme COPPA)
  • Google Analytics : Pour l'analyse d'utilisation (données anonymisées)

4.2 Cas exceptionnels

Nous pouvons partager des informations uniquement dans les cas suivants :

  • Obligation légale ou demande des autorités compétentes
  • Protection de la sécurité des utilisateurs
  • Prévention de la fraude ou d'activités illégales

5. Protection des données

5.1 Mesures de sécurité

  • Chiffrement : Toutes les données sont chiffrées en transit et au repos
  • Accès limité : Seuls les développeurs autorisés peuvent accéder aux données
  • Audits réguliers : Vérification périodique de la sécurité
  • Stockage local : Les données sont principalement stockées sur l'appareil

5.2 Conformité réglementaire

  • RGPD : Conformité avec le Règlement Général sur la Protection des Données
  • COPPA : Respect de la Children's Online Privacy Protection Act
  • Loi française : Conformité avec la législation française sur la protection des données

6. Droits des utilisateurs

6.1 Droits des parents/tuteurs

  • Accès : Demander une copie des données collectées
  • Rectification : Corriger les informations inexactes
  • Suppression : Demander la suppression des données
  • Portabilité : Récupérer les données dans un format lisible
  • Opposition : S'opposer au traitement des données

6.2 Exercice des droits

Pour exercer vos droits, contactez-nous à : privacy@kidaily.com

Nous répondrons à votre demande dans un délai de 30 jours maximum.

7. Conservation des données

7.1 Durées de conservation

  • Données d'évaluation : Conservées 3 ans maximum
  • Données d'utilisation : Conservées 2 ans maximum
  • Données publicitaires : Conservées 1 an maximum

7.2 Suppression automatique

Les données sont automatiquement supprimées à l'expiration des délais de conservation ou à la demande de l'utilisateur.

8. Cookies et technologies similaires

Notre application utilise des technologies similaires aux cookies pour :

  • Mémoriser les préférences de langue
  • Analyser l'utilisation de l'application
  • Personnaliser les publicités

Vous pouvez désactiver ces fonctionnalités dans les paramètres de votre appareil.

9. Modifications de cette politique

Nous pouvons modifier cette politique de confidentialité pour refléter les changements dans nos pratiques ou pour d'autres raisons opérationnelles, légales ou réglementaires.

En cas de modification importante, nous vous en informerons via :

  • Notification dans l'application
  • Email (si vous nous avez fourni votre adresse)
  • Mise à jour de cette page

11. Informations spécifiques aux enfants

Pour les parents et tuteurs :

Kidaily Light est conçue pour être utilisée par les parents et tuteurs avec leurs enfants. Nous ne collectons pas sciemment d'informations personnelles directement auprès d'enfants de moins de 13 ans sans le consentement vérifiable des parents.

Si vous découvrez que votre enfant nous a fourni des informations personnelles sans votre consentement, contactez-nous immédiatement à privacy@kidaily.com.
